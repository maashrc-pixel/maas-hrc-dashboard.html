## 🆘 TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| GitHub won't upload file | Try drag-drop instead of browse, or use smaller file size |
| Vercel shows 404 error | Make sure file name is exactly `maas-hr-dashboard.html` (not .html.html) |
| Dashboard loads blank | Refresh page (Ctrl+F5), clear cache (Ctrl+Shift+Delete) |
| Data not saving | Open console (F12), check for JavaScript errors |
| Deployment stuck | Go to Vercel dashboard, click Redeploy button |
| Can't find repository | Make sure it's "Public" not "Private" |
